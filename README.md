# Natours Landing Page

A modern, responsive landing page for adventurous tours. It features a responsive navigation, interactive tour cards, customer stories, and video backgrounds.

## Features
- Responsive navigation and header
- Tour cards with flip animations
- Customer testimonials section
- Background video section
- Call-to-action buttons for booking and exploring tours

## Tech
- HTML5
- Sass (SCSS)
- CSS3
- Google Fonts
- Custom icons
